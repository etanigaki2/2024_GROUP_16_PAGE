Detailed Description

The MainWindow class.

This class defines the main window of the application, responsible for the user interface, 3D rendering using VTK, and interaction with the VR rendering thread.

MainWindow constructor.

Constructs the MainWindow and sets up UI components and signal connections.

Constructs the main window.

Parameters
parentThe parent widget, defaults to nullptr.

This constructor initializes the main window, sets up the user interface using the provided ui object, establishes signal-slot connections for various UI elements, initializes the model part list and tree view, sets up the VTK rendering environment, and potentially starts the VR rendering thread.

Parameters
parentThe parent widget for this main window, typically nullptr for the top-level window.
